# QA 체크리스트 - 스마트스토어 키워드 최적화

## 📋 개요
이 문서는 스마트스토어 키워드 최적화 웹앱의 품질 보증을 위한 테스트 체크리스트입니다. 배포 전 모든 항목을 검증하여 안정적인 서비스를 제공합니다.

---

## 🧪 테스트 환경 설정

### 브라우저 테스트 환경
- [ ] Chrome 90+ (데스크톱)
- [ ] Chrome 90+ (모바일)
- [ ] Firefox 88+ (데스크톱)
- [ ] Safari 14+ (Mac)
- [ ] Safari (iOS)
- [ ] Edge 90+ (데스크톱)

### 화면 해상도 테스트
- [ ] 모바일: 360px ~ 414px
- [ ] 태블릿: 768px ~ 1024px
- [ ] 데스크톱: 1280px ~ 1920px
- [ ] 초대형: 2560px+

### 테스트 데이터 준비
- [ ] 샘플 키워드 CSV 파일 (정상 형식)
- [ ] 잘못된 형식의 CSV 파일
- [ ] 대용량 CSV 파일 (1000개 키워드)
- [ ] 특수문자 포함 키워드
- [ ] 한글/영문 혼합 키워드
