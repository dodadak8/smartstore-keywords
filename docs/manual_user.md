## 🚀 시작하기

### 서비스 개요
스마트스토어 키워드 최적화 도구는 판매자가 상품의 검색 노출을 향상시킬 수 있도록 도와주는 웹 기반 도구입니다.

### 주요 기능
- **키워드 분석**: 검색량과 경쟁도 기반 키워드 점수 계산
- **상품명 생성**: 최적화된 상품명 자동 생성 및 A/B 테스트
- **카테고리 추천**: AI 기반 최적 카테고리 제안
- **품질 점검**: 상품 등록 전 체크리스트 검증
- **데이터 관리**: CSV 내보내기/가져오기

### 첫 사용 가이드
1. [홈페이지](/) 접속
2. "키워드 리서치 시작하기" 클릭
3. 샘플 데이터로 기능 체험
4. 본격적인 키워드 분석 시작

## 🔍 키워드 리서치

### 시작하기
키워드 리서치는 모든 최적화 작업의 출발점입니다.

**접근 경로**: 홈 → 키워드 리서치 또는 `/keywords`

### 키워드 입력 방법

#### 1. 샘플 데이터 사용
처음 사용자라면 "샘플 데이터 추가" 버튼을 클릭하여 예시 데이터로 시작하세요.

**포함된 샘플 키워드**:
- 스마트폰 (검색량: 10,000, 경쟁도: 85%)
- 갤럭시 (검색량: 8,500, 경쟁도: 90%)
- 아이폰 (검색량: 12,000, 경쟁도: 95%)
- 무선이어폰 (검색량: 6,000, 경쟁도: 70%)
- 게이밍마우스 (검색량: 3,000, 경쟁도: 60%)

#### 2. CSV 파일 업로드
대량의 키워드를 한 번에 등록할 때 사용합니다.

**CSV 형식**:
```csv
term,volume,competition,score,tags,notes
"스마트폰",10000,85,75.2,"trending,category","모바일 디바이스"
"갤럭시",8500,90,68.9,"brand,trending","삼성 브랜드"
```

**필수 컬럼**:
- `term`: 키워드명
- `volume`: 월간 검색량
- `competition`: 경쟁도 (0-100)

**선택 컬럼**:
- `score`: 기회지수 (자동 계산 가능)
- `tags`: 태그 (쉼표로 구분)
- `notes`: 메모

#### 3. 수동 입력
개별 키워드를 직접 입력하고 싶다면 향후 버전에서 지원 예정입니다.

### 키워드 분석

#### 기회지수 계산
"점수 계산하기" 버튼을 클릭하면 각 키워드의 기회지수가 계산됩니다.

**계산 공식**:
```
기회지수 = (정규화된 검색량 × 0.7 + 태그 가중치 × 0.1) / (정규화된 경쟁도 × 0.3 + 1) × 100
```

**점수 해석**:
- 🟢 **70점 이상**: 매우 우수한 키워드
- 🟡 **50-69점**: 좋은 키워드  
- 🔴 **50점 미만**: 개선이 필요한 키워드

#### 필터링 및 정렬
검색창과 정렬 옵션을 사용하여 원하는 키워드를 빠르게 찾을 수 있습니다.

**정렬 기준**:
- 기회지수 (높은 순 권장)
- 검색량
- 경쟁도
- 키워드명 (가나다순)
- 등록일

### 키워드 태그 활용
태그를 통해 키워드를 분류하고 관리할 수 있습니다.

**태그 종류**:
- `trending`: 트렌드 키워드 (가중치 +20%)
- `brand`: 브랜드명 (가중치 +10%)
- `category`: 카테고리명 (기본)
- `feature`: 기능/특징 (가중치 -10%)
- `longtail`: 롱테일 키워드 (가중치 +30%)
- `seasonal`: 계절성 키워드 (가중치 -20%)

## ✏️ 상품명 생성

### 시작하기
키워드 분석이 완료되면 최적화된 상품명을 생성할 수 있습니다.

**접근 경로**: 키워드 리서치 → 상품명 생성하기 또는 `/titles`

### 키워드 선택
1. 등록된 키워드 목록에서 원하는 키워드들을 선택
2. **권장**: 기회지수 70점 이상 키워드 우선 선택
3. **다양성**: 브랜드, 카테고리, 특징 키워드를 균형있게 선택

### 상품 정보 입력

#### 필수 정보
- **카테고리**: 상품의 기본 분류 (예: 스마트폰, 의류, 화장품)
- **타겟 고객**: 주요 구매층 (예: 남성, 여성, 20대, 중년)

#### 선택 정보  
- **용도/시즌**: 사용 목적이나 계절성 (예: 겨울용, 운동용, 일상용)
- **특징/재질**: 상품의 주요 특징 (예: 방수, 무선, 고화질)

### 상품명 생성 과정
1. "상품명 생성하기" 버튼 클릭
2. 5-10개의 상품명 후보가 자동 생성됨
3. 각 후보별로 점수와 개선사항 확인

### 생성 결과 해석

#### 점수 체계
- **80점 이상**: 🟢 우수한 상품명
- **60-79점**: 🟡 양호한 상품명
- **60점 미만**: 🔴 개선 필요

#### A/B 띄어쓰기 테스트
각 상품명에 대해 두 가지 버전을 제공합니다:
- **띄어쓰기 버전**: "갤럭시 무선이어폰 블루투스 5.0"
- **붙여쓰기 버전**: "갤럭시무선이어폰블루투스5.0"

**선택 가이드**:
- 가독성 중시 → 띄어쓰기 버전
- 검색 최적화 중시 → 붙여쓰기 버전
- A/B 테스트로 성과 비교 권장

#### 개선 사항
각 상품명에 대한 구체적인 개선 방향을 제시합니다:
- 길이 조정 필요
- 키워드 위치 변경 권장
- 가독성 개선 필요

### 상품명 활용
1. **복사**: 마음에 드는 상품명을 클릭 한 번으로 복사
2. **편집**: 복사 후 필요에 따라 수동 편집
3. **테스트**: 여러 버전으로 등록하여 성과 비교

## 🏷️ 카테고리 추천

### 시작하기
적절한 카테고리 선택은 상품 노출에 매우 중요합니다.

**접근 경로**: 상품명 생성 → 카테고리 추천받기 또는 `/category`

### 추천 받기

#### 1. 키워드 선택
상품명 생성에서 사용한 키워드를 그대로 활용하거나 새로 선택할 수 있습니다.

#### 2. 추가 정보 입력 (선택사항)
더 정확한 추천을 위해 추가 정보를 입력할 수 있습니다:
- **타겟 고객**: 구매 고객층
- **용도/시즌**: 상품 사용 목적
- **주요 특징**: 핵심 기능이나 재질

#### 3. 추천 실행
"카테고리 추천받기" 버튼을 클릭하면 최대 3개의 카테고리 후보를 제시합니다.

### 추천 결과 분석

#### 신뢰도 해석
- **80% 이상**: 🟢 매우 적합한 카테고리
- **60-79%**: 🟡 적합한 카테고리
- **60% 미만**: 🔴 재검토 필요

#### 추천 근거 확인
각 카테고리별로 왜 추천되었는지 구체적인 이유를 제공합니다:
- 키워드 매칭 분석
- 패턴 인식 결과
- 유사 상품 분석 결과

#### 매칭 키워드
선택한 키워드 중 해당 카테고리와 일치하는 키워드들을 표시합니다.

#### 필수 속성 확인
선택한 카테고리에서 요구하는 필수 입력 속성들을 미리 확인할 수 있습니다.

### 카테고리 선택 가이드

#### 선택 우선순위
1. **신뢰도 80% 이상** 카테고리 우선 고려
2. **필수 속성** 모두 입력 가능한지 확인
3. **경쟁 상품** 카테고리 참고
4. **구체적인 카테고리** 선택 (너무 광범위한 카테고리 지양)

#### 주의사항
- 잘못된 카테고리 선택 시 검색 노출 감소
- 여러 카테고리가 가능해도 가장 구체적인 것 선택
- 카테고리 정책 변경에 따른 업데이트 필요

## ✅ 품질 점검

### 시작하기
상품 등록 전 최종 품질을 검증하는 단계입니다.

**접근 경로**: 카테고리 추천 → 품질 점검하기 또는 `/checklist`

### 자동 검증
"자동 검증 실행" 버튼을 클릭하면 등록된 데이터를 기반으로 자동 체크가 진행됩니다.

**자동 검증 항목**:
- 키워드 개수 (3개 이상 권장)
- 키워드 점수 (70점 이상 키워드 존재 여부)
- 키워드 다양성 (브랜드, 카테고리, 특징 키워드 균형)

### 수동 체크리스트

#### 기본정보 (필수)
- ✅ **키워드 리서치 완료**: 최소 3개 이상 키워드 등록
- ✅ **상품명 생성 완료**: 최적화된 상품명 선택
- ✅ **카테고리 선택 완료**: 적절한 카테고리 선택

#### 키워드 최적화 (권장)
- ⭐ **키워드 점수 70점 이상**: 주요 키워드의 기회지수 확인
- ⭐ **키워드 다양성**: 다양한 유형의 키워드 조합

#### 상품명 최적화 (권장)
- ⭐ **상품명 길이 적정성**: 20-50자 범위 권장
- ✅ **핵심 키워드 포함**: 상품명에 주요 키워드 포함
- ⭐ **가독성 및 자연스러움**: 읽기 쉬운 상품명

#### 카테고리 및 속성 (필수)
- ✅ **카테고리 정확성**: 상품과 정확히 일치하는 카테고리
- ✅ **필수 속성 입력**: 카테고리 요구 속성 모두 입력

#### 최적화 (권장)
- ⭐ **SEO 최적화**: 검색 엔진 최적화 고려
- ⭐ **모바일 최적화**: 모바일 환경 고려

### 점수 체계

#### 전체 점수 계산
- **필수 항목**: 20점씩 (총 100점 가능)
- **권장 항목**: 10점씩 (총 60점 가능)
- **최대 점수**: 160점 → 100점으로 환산

#### 점수별 가이드
- **80점 이상**: 🎉 우수한 품질, 등록 권장
- **60-79점**: 👍 양호한 품질, 일부 개선 후 등록
- **60점 미만**: ⚠️ 품질 개선 필요, 재작업 권장

### 개선 사항
체크리스트 결과를 바탕으로 구체적인 개선 방향을 제시합니다.

**개선 우선순위**:
1. **필수 항목** 먼저 완료
2. **권장 항목** 중 쉬운 것부터
3. **점수 대비 효과**가 큰 것 우선

## 💾 데이터 관리

### 내보내기

#### 키워드 CSV 내보내기
1. `/export` 페이지 접속
2. 프로젝트명 입력
3. "현재 데이터 로드" 클릭
4. "키워드 CSV 다운로드" 클릭

**활용 방법**:
- 엑셀에서 추가 분석
- 팀원과 데이터 공유
- 백업 목적

#### 전체 프로젝트 내보내기
JSON 형식으로 키워드, 상품명, 카테고리 등 모든 데이터를 포함하여 내보냅니다.

**활용 방법**:
- 프로젝트 백업
- 다른 환경으로 이전
- 버전 관리

### 가져오기

#### CSV 키워드 가져오기
1. CSV 파일 준비 (샘플 파일 다운로드 권장)
2. "파일 선택"에서 CSV 파일 업로드
3. 자동으로 키워드 가져오기 진행

**주의사항**:
- UTF-8 인코딩 필수
- 중복 키워드 자동 제거
- 필수 컬럼(term, volume, competition) 확인

#### JSON 프로젝트 가져오기
전체 프로젝트 파일을 업로드하여 이전 작업을 복원할 수 있습니다.

### 데이터 백업 전략

#### 정기 백업
- **일간**: 작업 종료 시 키워드 CSV 내보내기
- **주간**: 전체 프로젝트 JSON 내보내기
- **월간**: 중요 프로젝트 별도 보관

#### 협업 가이드
1. **팀 리더**: 마스터 데이터 관리
2. **팀원**: 개별 키워드 분석 후 CSV 공유
3. **취합**: 정기적으로 데이터 통합
4. **배포**: 최종 결과물 팀 공유

## 🔧 문제 해결

### 일반적인 문제

#### Q: 키워드 점수가 계산되지 않아요
**A**: 다음을 확인해보세요:
1. 검색량과 경쟁도 데이터가 올바르게 입력되었는지 확인
2. "점수 계산하기" 버튼을 클릭했는지 확인
3. 브라우저를 새로고침하고 다시 시도

#### Q: CSV 파일 업로드가 실패해요
**A**: 파일 형식을 확인해보세요:
1. 샘플 CSV 파일을 다운로드하여 형식 확인
2. UTF-8 인코딩으로 저장했는지 확인
3. 필수 컬럼(term, volume, competition)이 있는지 확인
4. 파일 크기가 5MB 이하인지 확인

#### Q: 상품명이 생성되지 않아요
**A**: 다음을 확인해보세요:
1. 키워드를 최소 1개 이상 선택했는지 확인
2. 상품 정보(카테고리, 타겟 고객)를 입력했는지 확인
3. 선택한 키워드에 점수가 계산되어 있는지 확인

#### Q: 데이터가 사라졌어요
**A**: 브라우저 설정을 확인해보세요:
1. 브라우저 캐시가 삭제되었을 가능성

### 효과적인 키워드 전략

#### 키워드 조합 전략
1. **핵심 키워드**: 상품의 핵심 기능/특징
2. **브랜드 키워드**: 브랜드명 또는 제조사
3. **롱테일 키워드**: 구체적이고 긴 키워드 문구
4. **계절/이벤트 키워드**: 시기적 관련성

#### 예시: 무선이어폰 상품
```
핵심: 무선이어폰, 블루투스이어폰
브랜드: 애플, 삼성, 소니
롱테일: 운동용무선이어폰, 게이밍블루투스이어폰
계절: 여름운동, 겨울러닝
```
