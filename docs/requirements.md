# 스마트스토어 키워드 최적화 웹앱 - 요구사항 명세서

## 📋 프로젝트 개요

### 목표
스마트스토어 판매자를 위한 "키워드 기반 상품명/카테고리 최적화 + 운영 체크리스트" 웹앱을 저비용으로 구현하여 판매자의 상품 노출도 향상을 지원합니다.

### 핵심 가치
- **저비용 운영**: 무료/극저가 호스팅으로 시작 가능
- **사용자 친화적**: 비전공자도 쉽게 사용할 수 있는 직관적 인터페이스
- **실용성**: 실제 스마트스토어 운영에 바로 적용 가능한 기능

## 🎯 핵심 기능

### 1. 키워드 리서치 보드
**목적**: 키워드 분석을 통한 최적의 키워드 선정

**기능 상세**:
- CSV 파일 업로드 또는 수동 키워드 입력
- 키워드별 데이터 필드:
  - 검색량 (수동 입력)
  - 경쟁도 (수동 입력, 0-100%)
  - CTR 가중치 (선택사항)
  - 태그 (계절, 이벤트, 롱테일 등)
  - 메모
- 기회지수 계산: `(검색량 × 가중치) / (경쟁도 + 1)`
- 정렬/필터링 기능
- 상위 N개 추천 키워드 제공

**입력 예시**:
```
키워드: "무선이어폰"
검색량: 6000
경쟁도: 70
태그: ["feature", "trending"]
메모: "블루투스 이어폰"
```

### 2. 상품명 생성기
**목적**: 스마트스토어 최적화된 상품명 자동 생성

**템플릿 구조**:
```
[카테고리] [성별/연령] [핵심키워드 1~3] [특징/재질/사이즈] [용도/시즌]
```

**주요 규칙**:
- 핵심키워드를 앞쪽에 배치
- 금칙어/중복어 자동 제거
- 글자 수 제한 (기본 60자) 표시
- 5~10가지 후보명 생성
- 띄어쓰기/붙여쓰기 A/B 버전 제공

**출력 예시**:
```
1. "갤럭시 무선이어폰 블루투스 5.0 고음질 운동용"
   - 띄어쓰기: "갤럭시 무선이어폰 블루투스 5.0 고음질 운동용"
   - 붙여쓰기: "갤럭시무선이어폰블루투스5.0고음질운동용"
   - 점수: 85.2점 (길이: 28자)
```

### 3. 카테고리/속성 추천
**목적**: 입력 키워드 기반 최적 카테고리 제안

**기능**:
- 키워드 패턴 분석을 통한 카테고리 매칭
- 3개 후보 카테고리 제시
- 각 후보별 추천 근거 제공
- 신뢰도 점수 (0-100%)
- 필수 속성 체크리스트
- 누락 속성 경고

**출력 예시**:
```
추천 카테고리: "디지털기기 > 이어폰/헤드폰"
신뢰도: 92%
근거:
- "무선이어폰" 키워드 매칭
- "블루투스" 기술 특징 확인
- 유사 상품 카테고리 분석
필수 속성: [연결방식, 배터리시간, 방수등급]
```

### 4. 체크리스트 & 품질 점검
**목적**: 상품 등록 전 최종 품질 검증

**체크 항목**:
- **기본정보**: 키워드 등록, 상품명 생성, 카테고리 선택
- **키워드 최적화**: 점수 70점 이상, 키워드 다양성
- **상품명 최적화**: 길이 적정성, 핵심키워드 포함, 가독성
- **카테고리**: 정확성, 필수 속성 입력
- **고급 최적화**: SEO, 모바일 최적화

**점수 시스템**:
- 필수 항목: 20점
- 권장 항목: 10점
- 전체 100점 만점
- 80점 이상 권장

### 5. 내보내기/불러오기
**목적**: 데이터 백업 및 협업 지원

**기능**:
- CSV 형식 키워드 내보내기
- JSON 형식 프로젝트 전체 내보내기
- CSV/JSON 파일 가져오기
- 샘플 템플릿 제공
- 프로젝트 단위 저장/불러오기

**CSV 형식**:
```csv
term,volume,competition,score,tags,notes
"스마트폰",10000,85,75.2,"trending,category","모바일 디바이스"
```

## 🔧 알고리즘 명세

### 키워드 점수 계산
```typescript
// 기회지수 계산 공식
score = (normalized_volume * weight_volume + tag_weight * weight_tag) / (normalized_competition + 1)

// 기본 가중치
weights = {
  volume: 0.7,    // 검색량 가중치
  competition: 0.3, // 경쟁도 가중치  
  tag: 0.1        // 태그 가중치
}
```

### 상품명 생성 규칙
1. **키워드 배치**: 핵심키워드 최소 1개 앞부분 배치
2. **정제 과정**: 
   - 금칙어 제거
   - 중복 단어 제거
   - 불필요 반복 기호 제거
3. **길이 검증**: 60자 초과 시 경고
4. **품질 점수**: 키워드 포함도, 가독성, 길이 적정성 종합

### 카테고리 추천 알고리즘
```typescript
// 카테고리 매칭 점수
categoryScore = keywordMatchScore * 0.4 + 
                patternMatchScore * 0.3 + 
                frequencyScore * 0.3

// 신뢰도 계산
confidence = Math.min(categoryScore * 100, 95) // 최대 95%
```
